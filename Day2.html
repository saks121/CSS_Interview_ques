1.What is box sizing?

The box-sizing property allows us to include the padding and border in an element's total width and height.

2.What is animation delay?

The animation-delay property specifies a delay for the start of an animation. The animation-delay value is defined in seconds (s) or milliseconds (ms).

4.What is grid-template-columns used for?

The grid-template-columns property specifies the number (and the widths) of columns in a grid layout. The values are a space separated list, where each value specifies the size of the respective column.
